# Tractive System ON (TS ON) PCB

## Overview
The TS ON circuit is designed such that the accumulator begins charging only when the shutdown circuit is closed and the TS ON button is pressed. If the shutdown circuit is open, charging is prevented even when the TS ON button is activated, ensuring safe charging operation of the accumulator charging system.
